# dnfeec

# Equip EXP Calculator for DNF 

# dnfeec是一个模拟DNF 105级装备成长/转移的工具
# 测试现成长经验偏移误差为0.1%

#主要功能如下：
    创建装备
    随机装备
    装备成长
    经验转移
    本地保存背包
    读取本地背包
    清空本地内容
    
    
#本项目使用vue3.0 + element-plus构建

#使用前请使用npm install / yarn install

#打包方式如下

#1. 使用vue打包,运行 build 打包成html
#2. 使用electron打包,运行 electron:build 打包成PCexe
